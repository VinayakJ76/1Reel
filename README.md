# Cinematic Reel Maker (Android)

A starter Android Studio project for creating cinematic social reels from:
- **Live camera captures**
- **Videos and photos from gallery**
- **Audio files from device storage**

## Features implemented
- Jetpack Compose UI with a cinematic, creator-focused dashboard.
- "Shoot Clip" flow using Android camera capture contract.
- "Import Media" flow for photos/videos from device.
- "Import Audio" flow for audio tracks from storage.
- Creative preset ideas (Beat Sync Cuts, LUT moods, AI story flow, velocity ramps, caption pulse).
- Timeline list showing selected assets.

## Build in Android Studio
1. Open this folder in Android Studio.
2. Let Gradle sync complete.
3. Run on an Android device/emulator (API 26+).

> Camera capture requires camera hardware and camera permission.
